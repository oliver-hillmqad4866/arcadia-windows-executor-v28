## About Arcadia

Arcadia is a Windows-focused Roblox script executor built to run Lua scripts from a desktop environment. Its workflow emphasizes rapid execution, a clean layout, and a script hub that keeps frequently used Roblox scripts easy to reach.

The app is aimed at people looking for a practical Roblox executor with auto-update support, multilingual UI options, and a persistent workspace for script handling. In addition to one-click execution, Arcadia offers batch execution and a built-in debugger to make script testing more structured.
